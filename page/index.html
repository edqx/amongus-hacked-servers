You can choose options for your Among Us servers that you can't normally choose. Some are limited however, e.g. the player count, because Among Us rejects your server create request with any more. Others are also maxed at 255, as this is the max value allowed for unsigned chars (which is what the game uses for these values.). To install your host options, click download and replace the file with the same name in "Appdata\LocalLow\Innersloth\Among Us" with the file that you downloaded.
